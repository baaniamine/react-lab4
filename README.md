# react-lab4

Application React de type To-Do List interactive.

## Fonctionnalites

- ajout de taches
- marquage des taches comme terminees
- suppression des taches
- interface stylisee avec ressource image depuis `public/`

## Demarrage

```powershell
npm install
$env:PORT=3000
npm.cmd start
```
